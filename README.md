# WS タイトルカウンター

ヴァイスシュヴァルツの大会で、参加者が使用しているデッキの**タイトルをタップで集計**し、
**集計表**と**円グラフ**を SNS 投稿用の画像として書き出せる、インストール不要の単体 HTML アプリです。
スマホ・タブレットでそのまま動き、カウントは端末内に自動保存されます。

➡ **デモ（GitHub Pages 公開後）**: https://Yukikko-ktrk.github.io/WS-Title-Counter/
（リポジトリ名に合わせて URL は調整してください）

## 特徴

- 📋 タイトル一覧をタップして使用デッキをカウント。右端の **50音インデックス**で素早くジャンプ
- 🔢 **集計表**（使用数・降順／同率タイトルは1行にまとめて省スペース表示）
- 🥧 **円グラフ**（使用率の高いタイトルほど大きいフォント／しきい値スライダーで表示するタイトルを調整）
- 🖼 集計表・円グラフを **PNG 画像**で書き出し（SNS 投稿向け・白背景）
- 💾 カウントはブラウザに**自動保存**（オフライン可）／**ライト・ダークテーマ**切替
- 🔄 公式カードリストからタイトル一覧を更新するスクリプト同梱

## 使い方

`index.html` をブラウザで開くだけです（スマホ・タブレット可）。画面は3つのタブに分かれています。

1. **カウント入力**
   - タイトル名をタップするたびに使用数 +1（押し間違いは各行の「−」で減算）。
   - 上部の検索で絞り込み、並びは **50音順 / カウント順**（カウント済みを上に降順）を切替。
   - 右端の50音バーで各行へジャンプ。
2. **集計・画像出力**
   - 集計表（使用数・降順、同率は1行）を表示。
   - 「集計表を画像保存」「円グラフを画像保存」「2枚まとめて保存」で PNG をダウンロード。
   - 「円グラフの作成」のスライダーで **「何人以上に使われたタイトルを個別表示するか」**を決め、ボタンで反映（それ未満は「その他」に集約）。
3. **設定**
   - 大会名（出力画像に表示）、参加者数 / タイトル数、テーマ（ライト・ダーク）、カウントの全リセット。

> カウントはこの端末のブラウザ（localStorage）にのみ保存され、サーバには送信されません。

## ファイル構成

| ファイル | 役割 |
|---|---|
| `index.html` | アプリ本体（これだけで動作） |
| `titles.js` | タイトル一覧データ（`window.WS_TITLES`） |

## タイトル一覧の更新

`titles.js` は公式カードリスト（ws-tcg.com）から取得したタイトル一覧を焼き込んだものです。
新弾などでタイトルが増えたら、メンテナンス用のローカルスクリプトで `titles.js` を再生成して更新します。

> ブラウザの CORS 制約のため、アプリ実行中に ws-tcg.com を直接読みに行くことはできません。
> そのため、あらかじめ `titles.js` に焼き込む方式を採用しています。
> （更新用スクリプトはローカル管理のため、このリポジトリには含めていません。）

## GitHub Pages での公開

1. このフォルダを GitHub リポジトリに push。
2. リポジトリの **Settings → Pages** で、Branch を `main`（フォルダ `/ (root)`）に設定。
3. 発行された URL を開けば、インストール不要でスマホからも利用できます。

初回だけ git のセットアップが必要です。

```bash
git init
git remote add origin <リポジトリURL>
git add .
git commit -m "init"
git push -u origin main
```

以降は `update_and_push.bat` だけでタイトル一覧の更新と反映ができます。

## 動作環境

- 表示・カウント：モダンブラウザ（Chrome / Edge / Safari など）。
- タイトル一覧の更新：Windows（`update_titles.ps1` / `update_and_push.bat` は PowerShell・git を使用）。
